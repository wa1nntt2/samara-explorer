# Samara Explorer Backend

Бэкенд для мобильного приложения "Исследователь Самарской области".

## Описание
API для создания и просмотра интересных мест в Самарской области с фотографиями и геолокацией.

## Технологии
- Python 3.12
- FastAPI
- PostgreSQL + PostGIS (в Docker)
- SQLAlchemy
